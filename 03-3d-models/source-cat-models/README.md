# CatLife Cat Model Sources

This folder stores original local cat model source packages that should remain available for recovery or re-export.

## Contents

- `original-meshy-quadruped/`
  - Original Meshy low-poly orange cat quadruped package supplied for the animation task.
  - Includes the original extracted folder and the original `.zip` package.

## Usage

Use `03-3d-models/blender-work/CatLife_cat_animation_coordinate_corrected.blend` for current animation production. This source-model folder is for provenance and recovery, not the active working file.
