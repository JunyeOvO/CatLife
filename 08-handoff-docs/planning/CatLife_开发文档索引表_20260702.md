# CatLife 开发文档索引表

生成日期：2026-07-02

用途：给后续实际开发、Unity/Android 联调、LLM 代码包、PPT/视频/海报提交使用。此表不是资料包清单的替代品，而是“开发时先看哪份文档”的工作索引。

## 使用规则

| 规则 | 说明 |
|---|---|
| 优先读项目源文件 | `README.md`、`07-tech-specs/`、`08-handoff-docs/`、`03-3d-models/`、`06-deliverables/` 是开发判断来源。 |
| `12-docs-package/` 只作归档副本 | 该目录用于资料打包、查重和交接，不优先修改；同名内容应回到项目源位置维护。 |
| `work/` 只作运行证据和历史参考 | Unity 当前主工程在 `work/CatLife_Unity_Main/`；旧 `work/unity-android-build-batch-20260629/` 只读参考。 |
| 大二进制资产不默认入 Git | `.blend`、`.fbx`、`.glb`、视频、zip 等按 `AGENTS.md` 和 manifest 记录处理。 |
| 开发前先看 P0 入口 | 每次新任务先读“P0 开发入口”对应行，再按模块读取下级文档。 |
