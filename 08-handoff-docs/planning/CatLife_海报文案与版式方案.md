# CatLife 海报文案与版式方案

日期：2026-06-29
目标：按复赛官方要求准备 70cm x 150cm 竖版海报内容，避免最后只剩截图拼贴。

## 1. 官方规格

| 项 | 要求 |
|---|---|
| 方向 | 竖版 |
| 尺寸 | 70cm x 150cm |
| 格式 | jpg / jpeg / png |
| 文件大小 | PDF 文本抽取为“整体大小不低于 2M”；最终上传前按平台限制复核 |
| 必含内容 | 作品名称、宣传口号、宣传图 |

## 2. 推荐主视觉

主视觉优先级：

1. 猫咪小镇全景，能一眼看到低多边形风格。
2. 动画猫放在前景偏下位置，形成陪伴感。
3. 中心区域留给一句 slogan，不要把技术文字堆满。
4. 下方放 4 个状态小图标或真实截图：普通、过渡、专注、奖励。

## 6. 版式注意事项

- 不要把二维码、Logo、队名做得比作品名更抢眼。
- 不用大量小字解释技术，技术细节放 PPT。
- 小镇和猫必须是第一视觉中心。
- 海报上出现的画面应与最终视频/APK一致，避免评委看到“海报很好但产品里没有”。
- 如果最终截图还没完成，先用占位版本排版，等 APK/Game View 截图后替换。
