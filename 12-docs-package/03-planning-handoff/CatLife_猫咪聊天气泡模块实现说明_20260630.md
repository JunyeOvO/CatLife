# CatLife 猫咪聊天气泡模块实现说明

日期：2026-06-30
目标：在 CatLife 主小镇和专注流程中加入“猫咪冒出聊天气泡”的轻提示模块。气泡内容根据用户当前状态抛出提醒或鼓励，不做长聊天、不做活动页、不遮挡猫咪小镇场景。

## 1. 模块定位

```text
Scene Layer = 猫咪 + 猫咪小镇背景
UI Overlay = 顶部状态、功能按钮、摄像机按钮、开始专注按钮、猫咪聊天气泡
Cat Chat Bubble = 跟随猫咪锚点的小气泡，只显示状态提醒或鼓励
```

气泡不是完整聊天系统。它是“行为状态 -> 猫咪反馈”的可视化出口，用于把专注提醒表达成猫咪陪伴。

## 2. Unity 对象树

```text
CatLife_Main
├── TownRoot
├── CatRoot
│   └── BubbleAnchor
├── CameraRig
│   └── Main Camera
└── UIRoot
    └── Canvas
        └── MainTownPanel
            └── CatChatBubble
                ├── BubbleBackground
                └── BubbleText
```

`BubbleAnchor` 是猫模型下的空物体，放在猫头上方。猫移动或切动画时，气泡通过屏幕坐标跟随这个锚点。

## 3. 状态和文案

| 状态 | 触发 | 气泡文案 | 建议猫动画 |
|---|---|---|---|
| Normal | 进入小镇、普通陪伴、长时间无操作 | 先不用急，我在这里。 | IdleBreath |
| Transition | 用户行为变慢、准备进入专注 | 你慢下来了，我也安静一点。 | CuriousSniff / HeadTiltListen |
| Focus | 专注开始、稳定专注 | 我会轻轻陪着你，不打扰。 | IdleBreath |
| DistractionNudge | 行为识别分心分值较高 | 要不要回到刚才那件事？ | HeadTiltListen |
| Reward | 完成专注 | 完成啦，猫咪给你一个小爪印。 | TailWagHappy |

## 4. Unity 脚本

脚本位置：

```text
06-deliverables/catlife-ui-assembly-kit-20260629/unity-scripts/CatLifeCatChatBubbleController.cs
```

核心接口：

```csharp
ShowForState(CatLifeCatBubbleState.Normal);
ShowForState(CatLifeCatBubbleState.Transition);
ShowForState(CatLifeCatBubbleState.Focus);
ShowForState(CatLifeCatBubbleState.DistractionNudge);
ShowForState(CatLifeCatBubbleState.Reward);
ShowMessage("自定义短提示", 4.5f);
Hide();
```

推荐接线：

| 字段 | Unity 引用 |
|---|---|
| `rootCanvas` | `UIRoot/Canvas` |
| `worldCamera` | `CameraRig/Main Camera` |
| `catAnchor` | `CatRoot/BubbleAnchor` |
| `bubbleRoot` | `Canvas/MainTownPanel/CatChatBubble` |
| `bubbleText` | `CatChatBubble/BubbleText` |
| `canvasGroup` | `CatChatBubble` 上的 CanvasGroup |

## 5. UI 规格

| 项 | 标准 |
|---|---|
| 形态 | 半透明白色圆角气泡 + 小尾巴 |
| 位置 | 猫头上方或侧上方，跟随猫移动 |
| 面积 | 小于屏幕 8%，不遮猫脸、不挡主按钮 |
| 字数 | 6-18 个中文字符为主 |
| 显示时间 | 默认 4.5 秒 |
| 冷却 | 默认 8 秒，避免专注中连续刷屏 |
| 交互 | 不拦截点击，`blocksRaycasts = false` |

## 6. 行为识别接入

行为识别或状态机只需要在状态变化时调用气泡接口：

```text
普通状态进入 -> ShowForState(Normal)
专注准备阶段 -> ShowForState(Transition)
专注开始 -> ShowForState(Focus)
检测到高分心 -> ShowForState(DistractionNudge)
专注完成 -> ShowForState(Reward)
进入记录/隐私页 -> Hide()
```

后续如果接入大模型解释，气泡仍保持短句。LLM 输出只能经过安全改写后进入气泡，不能直接把长解释显示在猫旁边。

## 7. 装配步骤

1. 在 `CatRoot` 下创建 `BubbleAnchor`，放到猫头上方。
2. 在 `Canvas/MainTownPanel` 下创建 `CatChatBubble`。
3. 给 `CatChatBubble` 添加 `RectTransform`、`CanvasGroup`、背景 Image、`TMP_Text`。
4. 挂载 `CatLifeCatChatBubbleController`。
5. 把 `CatLifeCompetitionDemoFlow.chatBubble` 引用到该控制器。
6. 用 DemoFlow 验证 Normal、Transition、Focus、Reward 四类气泡。
7. 行为识别完成后，把 `DistractionNudge` 接到高分心事件。

## 8. 验收标准

| 编号 | 标准 |
|---|---|
| BUBBLE-01 | 气泡能显示在猫附近，而不是屏幕固定死角 |
| BUBBLE-02 | 猫移动时气泡跟随 `BubbleAnchor` |
| BUBBLE-03 | 气泡不会遮挡猫脸、底部开始按钮和右侧功能按钮 |
| BUBBLE-04 | Normal/Transition/Focus/Reward/DistractionNudge 都有明确短文案 |
| BUBBLE-05 | 气泡自动隐藏，且冷却时间内不重复刷屏 |
| BUBBLE-06 | 关闭 Canvas 后，场景仍只有猫咪和猫咪小镇背景 |
