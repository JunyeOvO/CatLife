# CatLife 用户验证访谈与问卷模板

日期：2026-06-29
目标：补齐复赛创新性和应用价值所需的“用户真实需求”证据，至少形成 5 份可进入 PPT 的访谈或问卷摘要。

## 1. 目标人群

| 人群 | 场景 |
|---|---|
| 大学生 | 期末复习、考研/考证、自习室、图书馆、宿舍学习 |
| 高中/研究生可选 | 作为补充样本，不替代大学生主样本 |

## 2. 样本要求

- 最低：5 份有效访谈或问卷。
- 推荐：8-12 份，筛选 5 份典型摘要进入 PPT。
- 记录：受访者匿名编号，不收集手机号、学号、真实姓名。
- 输出：只保留场景、痛点、现有工具、对 CatLife 的反馈。

## 3. 快速访谈提纲

每人 5-8 分钟即可。

1. 你通常在哪里学习或复习？
2. 开始学习前，最常被什么打断？
3. 你用过番茄钟、锁机软件或专注 App 吗？为什么继续用或不用？
4. 你更讨厌“强制锁住不能退出”，还是“没人提醒导致一直拖延”？
5. 如果有一只猫根据你的操作节奏从活跃变安静，你觉得有帮助吗？
6. 你能接受应用只统计本 App 内的点击/滑动节奏来判断状态吗？
7. 你不能接受哪些数据被采集？
8. 你希望完成专注后收到什么样的反馈？

## 4. 问卷题目

| 题号 | 题目 | 类型 |
|---|---|---|
| Q1 | 你的主要学习场景是？ | 多选：图书馆/宿舍/自习室/课堂间隙/其他 |
| Q2 | 开始学习前，你是否经常被手机分心？ | 1-5 分 |
| Q3 | 你最难的是开始专注、维持专注，还是分心后回归？ | 单选 |
| Q4 | 你用过哪些专注工具？ | 多选：番茄钟/锁机/宠物陪伴/未使用 |
| Q5 | 你放弃这些工具的原因是？ | 多选：太强制/太无聊/太复杂/没反馈/其他 |
| Q6 | 你是否接受“轻锁定”，即仍可退出但需要明确动作？ | 1-5 分 |
| Q7 | 猫咪随状态变安静是否会降低打扰感？ | 1-5 分 |
| Q8 | 你是否接受只上传聚合特征给模型生成反馈文案？ | 1-5 分 |
| Q9 | 你最不能接受被采集的数据是？ | 多选：输入内容/录屏/跨 App 行为/位置/都不能 |
| Q10 | 你希望 CatLife 最先做好哪项功能？ | 单选：猫咪陪伴/轻锁定/专注记录/大模型反馈/小镇成长 |

## 6. 可量化指标

进入 PPT 时建议统计：

- 经常被手机分心的比例；
- 最困难阶段：开始/维持/回归；
- 接受轻锁定的平均分；
- 接受猫咪安静陪伴的平均分；
- 隐私不可接受项排序。

## 7. 隐私和合规

- 不收集真实姓名、手机号、学号。
- 不展示原始问卷截图里的个人信息。
- PPT 中只放匿名编号和聚合统计。
- 如果引用原话，先去除可识别信息。
