# CatLife 主页目标稿拆解与摄像机控制

日期：2026-06-29
目标：以已确认的 CatLife 主页目标图为基准，拆成 Unity 可实现的场景层、UI 叠加层、猫咪活动层和摄像机控制层。

## 1. 已确认目标

```text
摄像机固定在猫咪前方正对猫；
没有 UI 时画面只有猫咪 + 猫咪小镇背景；
UI 是叠加在画面上的 Canvas 图层；
活动角色只有猫咪；
猫咪可在摄像机范围内走动；
摄像机可水平 360 度原地旋转，也可水平前后移动，高度不变。
```

## 2. 分层拆解

| 层 | Unity 对象 | 内容 | 规则 |
|---|---|---|---|
| Scene Layer | `TownRoot` | 猫咪小镇 3D 场景、道路、建筑、树、花、天空盒 | 不包含 UI，不包含其他活动角色 |
| Cat Layer | `CatRoot` | 橘猫模型、Animator、行为驱动 | 可走动；不依赖坐下动画 |
| Camera Layer | `CameraRig/Main Camera` | 固定高度的正面猫视角 | 只允许水平旋转和前后移动 |
| UI Overlay | `Canvas` | 顶部状态、右侧按钮、底部开始专注、摄像机按钮 | 叠加层，不阻挡 80% 以上场景 |
| Logic Layer | `Managers` | 状态机、行为识别、LLM 降级、UI 切换 | 不直接改变场景层结构 |

## 3. 猫咪动画约束

当前不能假设有“猫咪坐下”动画。

| 状态 | 可用动作策略 |
|---|---|
| Normal | 猫在摄像机范围内慢速走动、IdleBreath、看向镜头 |
| Transition | CuriousSniff、HeadTiltListen、靠近或短暂停顿 |
| Focus | HeadTiltListen、IdleBreath、低频呼吸/轻摆尾；不强制坐下 |
| Reward | TailWagHappy、伸懒腰/开心动作，如后续动画可用再替换 |

## 4. 猫咪活动范围

建议把猫限制在摄像机前方的一个局部矩形内：

```text
CatLocalBounds:
  X: -1.6 ~ 1.6
  Z: -0.8 ~ 2.4
  Y: 保持地面高度
```

## 5. 摄像机控制按钮

主页 Overlay 需要 4 个摄像机控制按钮：

| 按钮 | 行为 | 高度 |
|---|---|---|
| 左旋 | `CameraRig` 原地绕世界 Y 轴向左旋转 | 不变 |
| 右旋 | `CameraRig` 原地绕世界 Y 轴向右旋转 | 不变 |
| 前进 | 沿摄像机水平 forward 前进 | 不变 |
| 后退 | 沿摄像机水平 forward 后退 | 不变 |

禁止：

- 上下移动；
- 俯仰自由旋转；
- roll；
- 双指缩放；
- 顶视角切换；
- 让摄像机离开猫咪活动范围。

实现脚本：

```text
06-deliverables/catlife-ui-assembly-kit-20260629/unity-scripts/CatLifeFixedHeightCameraController.cs
06-deliverables/catlife-ui-assembly-kit-20260629/unity-scripts/CatLifeCameraButtonHold.cs
```

## 6. UI Overlay 元素

| 区域 | 元素 | 规则 |
|---|---|---|
| 左上 | CatLife、今日专注时间 | 文字悬浮，不加大卡片 |
| 右侧 | 猫咪、记录、设置按钮 | 圆形按钮，贴边，不遮猫脸 |
| 左下 | 左旋、右旋、前进、后退 | 小圆按钮，保持低遮挡 |
| 底部 | 开始专注 | 主按钮，只占底部，不挡猫主体 |
| 底部文字 | 状态文案 | 单行，避免卡片化 |

## 7. Unity 接线

```text
CameraRig
  CatLifeFixedHeightCameraController
    cameraRig = CameraRig
    cameraTransform = Main Camera

Canvas/CameraControls/RotateLeftButton
  CatLifeCameraButtonHold
    onHoldStart -> CameraRig.HoldRotateLeft
    onHoldEnd -> CameraRig.StopRotate

Canvas/CameraControls/RotateRightButton
  CatLifeCameraButtonHold
    onHoldStart -> CameraRig.HoldRotateRight
    onHoldEnd -> CameraRig.StopRotate

Canvas/CameraControls/MoveForwardButton
  CatLifeCameraButtonHold
    onHoldStart -> CameraRig.HoldMoveForward
    onHoldEnd -> CameraRig.StopMove

Canvas/CameraControls/MoveBackButton
  CatLifeCameraButtonHold
    onHoldStart -> CameraRig.HoldMoveBack
    onHoldEnd -> CameraRig.StopMove
```

## 8. 验收标准

| 编号 | 标准 |
|---|---|
| HOME-01 | 关闭 Canvas 后，Game View 只有猫咪和猫咪小镇背景 |
| HOME-02 | 打开 Canvas 后，UI 只是叠加按钮，不出现大遮挡卡片 |
| HOME-03 | 猫咪能在镜头范围内走动，且不会走出画面主体区域 |
| HOME-04 | 左旋/右旋能 360 度水平原地旋转，高度不变 |
| HOME-05 | 前进/后退只改变水平位置，高度不变 |
| HOME-06 | 行为识别状态能驱动猫动画，但不依赖坐下动画 |
