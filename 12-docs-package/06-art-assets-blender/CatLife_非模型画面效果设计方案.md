# CatLife 非模型画面效果设计方案

同步来源：`10-art-guide/CatLife_非模型画面效果设计方案.md`

本资料包副本用于交接阅读。后续维护请优先修改项目源文档，再同步到资料包。

当前权威版：`10-art-guide/CatLife_非模型画面效果深化方案.md`。

核心结论：

- 主视觉方向：晴朗午后专注花园。
- 优先补齐：天空盒、暖向主光、Global Volume、低密度状态粒子、blob/contact shadow。
- 避免：写实 HDRI、暗夜强发光、重景深、运动模糊、浓雾、大面积半透明烟雾。
- 移动端默认预算：同屏粒子 60-90，Bloom + Color Adjustments + Vignette 三项以内，主光 Shadow Distance 25-35，Additional Light Shadows 关闭。
